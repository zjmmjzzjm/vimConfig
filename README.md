# vimConfig
my vim configuration
* Before pulling, just a few steps should be taken: 
  * cd ~; 
  * mkdir .vim; 
  * cd .vim; 
  * git init; 
  * git remote add *This repository*
* After the repository be pulled, you should execute commands :
  * git submodule init
	git submodule update
  * vim +VundleInstall +qa!
